Raven Paint and Animation

This is the home of Raven, a vector editing and animation suite.

Things are still pretty rough right now. I am in the process of uploading everything and making sure it's running properly. Check back frequently to see any changes.

You can browse the source code on the project page.

API Docs

Raven Editor - Contains the editor itself and the main scenegraph used by the Raven project.

Raven Core - Core data structures and utilities that allow the creation and use of Raven scenes.

Coyote Graphics - A fast, platform independent library for rendering in Java. Built to be compatible with the OpenGL ES 2.0 standard.

Raven Docking - A window docking system that allows multiple Swing components to share the same container without overlapping.

Raven Service - A handy tool that uses annotations to identify and create indexes for classes that will be used as services.

SVG Rabbit - An under development SVG parser/engine used to import and export data.

SWF Parser - A parser for the SWF file format.
