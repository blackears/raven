Overview

Raven Docking is a docking window manager used by the Raven Studio project. You can use it to easily lay out multiple windows in a single container.

 

Run with webstart

 

Raven Docking will allow you to divide your main window into multiple smaller regions that can be resized simeltaneously by dragging the region borders. You may also place multiple windows in the same area and use tabs to select between them. Best of all, the user can easily drag and drop windows between regions, providing a natural way to customize their workspace.
