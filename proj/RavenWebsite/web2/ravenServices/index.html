Overview

Ravens Services is a tiny utility library that makes it easier to create and maintain lists of Java services.

What are services?

Raven is designed around a plugin architecture. Plugins provide an easy way to break up a large project into several smaller, independent projects. Plugins also allow third party developers to extend your application without you having to write a single extra line of code. Java provides plugin functionality via its services API.

What does Raven Services do?

At the heart of Java services is the services list. Every jar which provides services must maintain lists of the services they provide in the META-INF/service subdirectory within their jar. This can be tedious to do by hand, and is hard to maintain.

Raven Services uses annotations and the annoation processor to let you mark and automatically generate lists of services whenever you compile your Java source code. It also keeps track of what has already been processed so that you can incrementally compile your code.
