Current Version <pre-release>

I am currently in the process of migrating Raven from a previous repository on another server and making lots of little bug fixes to make it ready for presentation.

The current version of Raven is <pre-release>. That's right, it doesn't even have a version yet. When it's all ready to go, I'll launch it with its first version number.

At the moment, there is a lot of heavy lifting going on behind the scenes, with features being upgraded. When the below list is complete, I'll formally make a release, hopefully before spring 2012:

  • Finish replacing Java2D with OpenGL ES 2.0
  • Fix bugs in path/mesh geometry
  • Update file format to handle subprojects
  • Finish node editor view
  • Adapt SWF importer to new code
  • Finish SVG importer
