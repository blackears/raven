Raven Studio

Overview

Coyote Graphics is the core graphics engine of Raven Studio. It is also a library of math functions used for building and modifying shapes.

Motivation

CoyoteGraphics is a platform independent 3D rendering pipeline for Java. It is based on the OpenGL ES 2.0 specification.

Raven Studio was originally written around the Java 2D API. However, this API proved to be too slow for realtime animation. It was also not possible to port Java2D to other Java platforms, such as Google Android. This prompted me to write an entirely new graphics API for Java, with both rendering speed and portability in mind.
