Paint Layout Tool

All paints have a layout that affects how they are displayed. The paint layout can stretch, rotate and otherwise change how the paint is drawn on-screen. This tool provides an easy way to manipulate the paint layout.

Some paints (such as flat colors) will not be visibly altered by their paint layout.

There are three different sub-tools for laying out paints.

  • Linear paint layout provides a simple line along which a linear gradient can be spread.
  • Radial paint layout makes laying out radial grandients easy. You can also set the gradient focus by holding <Ctrl> while clicking and dragging.
  • Texture paint layout is the easiest way to layout textures by manipulating the shape of the box a single texture unit will fill.
